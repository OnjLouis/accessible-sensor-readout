Por que existe Sensor Readout

Toda esa informacion sobre tu propio ordenador que antes era dificil de encontrar, escondida detras del Administrador de dispositivos o encerrada en apps caras o inaccesibles, deberia ser mas facil de alcanzar.

Sensor Readout busca reunir tanta de esa informacion como sea posible en una app gratuita, facil de usar, portable o instalable, para que no tengas que buscarla por todas partes. Estaba harto del estado de la accesibilidad en la parte tecnica de la informatica con Windows, asi que quise crear algo que yo mismo quisiera usar. Al parecer, muchas otras personas tambien querian eso.

Espero que descubras tanto sobre tu ordenador con esta app como yo he descubierto sobre el mio, y espero que te sorprenda gratamente lo que encuentres.

Mi deseo es que Sensor Readout se convierta en el referente de su clase: una herramienta pensada primero para la accesibilidad, que las personas ciegas y con baja vision que trabajan en soporte tecnico puedan entregar a sus clientes para que generen informes y reciban ayuda para resolver problemas. Busco la excelencia, y espero que lo notes al usarla.

Instalador de requisitos

Ejecute primero Install-Prerequisites.cmd desde esta carpeta. Llama a Install-Prerequisites.ps1, pide permisos de administrador si hace falta e instala PawnIO. Comandos de instalacion manual

Si el instalador de requisitos no funciona en su sistema, abra Windows Terminal o PowerShell como administrador y ejecute estos comandos:

Herramienta opcional de diagnostico:

winget install --id LibreHardwareMonitor.LibreHardwareMonitor -e --source winget

Preferencias

Use Opciones > Preferencias o Ctrl+,. El cuadro recuerda la pestana usada por ultima vez en la sesion actual.

General (Ctrl+1)

Idioma, actualizacion, intervalo, icono del area de notificacion, unidad de temperatura, separador decimal, nivel de registro, busqueda de actualizaciones y lecturas del area de notificacion.

Inicio e instalacion (Ctrl+2)

Instalacion en este PC, inicio con Windows, inicio minimizado, mensaje hablado de inicio, sonidos de inicio/cierre y feedback de diagnostico.

Instalar en este PC... copia la carpeta portable actual a la carpeta de programas de Windows de este usuario, puede crear un acceso directo en el escritorio, cierra la copia antigua e inicia la copia instalada. Si el inicio con Windows esta activado, el acceso directo de inicio apunta despues a la copia instalada.

Cuando Sensor Readout ya se ejecuta desde la ubicacion instalada, este boton cambia a Desinstalar de este PC.... La desinstalacion quita los archivos instalados y accesos directos, pero deja Config, Logs y Reports. La misma pestana tambien incluye una casilla de acceso directo del escritorio, para agregarlo o quitarlo sin reinstalar.

Si Sensor Readout se reinicia despues de instalar o actualizar y no aparece la ventana principal, compruebe si esta activado Iniciar minimizado al area de notificacion. Windows tambien puede ocultar el icono hasta que lo permita en la configuracion del area de notificacion o bandeja del sistema. Al mostrar el icono de Sensor Readout, podra abrir la ventana principal desde ese icono.

Atajos de teclado (Ctrl+3)

Mostrar/ocultar global, leer area de notificacion, perfiles hablados, copia por doble pulsacion y nombres hablados cortos.

Perfiles de ventilador (Ctrl+4)

Grupos de acciones de ventilador con hotkey opcional, modo conmutador, mensaje hablado y sonido.

Alarmas (Ctrl+5)

Vigilar lecturas, umbral, unidad, cooldown, voz y sonido.

Plug-Ins (Ctrl+6)

Ver, describir, importar y activar plug-ins de hardware instalados.

Elementos ocultos (Ctrl+7)

Restaurar lecturas o grupos ocultos desde la ventana principal.

La pestana tambien incluye la preferencia de confirmacion de instalacion de actualizaciones, para volver a activar el aviso de seguridad despues de ocultarlo.

Editor de idioma (Ctrl+8)

Editar archivos de idioma o crear uno nuevo desde ingles. Las cadenas que falten vuelven a ingles.

Si el estado del area de notificacion esta activado, minimizar oculta Sensor Readout de la barra de tareas y Alt+Tab. Abralo desde el icono. Alt+F4 sale de la app.

Las lecturas del area de notificacion se mueven de una lista disponible a una lista de orden. Windows muestra tooltips cortos, por eso el limite es cuatro. Una lectura aparece en una sola lista. Use botones o Ctrl+flechas. Los elementos muestran dispositivo, nombre y categoria, por ejemplo Ethernet - Rx: Red.

La pestana Atajos de teclado puede importar perfiles de atajos de teclado hablados desde otra configuracion de equipo. Los perfiles importados no conservan sus teclas antiguas, para no robar teclas ya usadas en este equipo. Informes

Los informes sirven para dos situaciones: guardar una instantanea de su propio equipo o enviar esa instantanea a otra persona para que pueda verla dentro de Sensor Readout.

Guardar un informe

Pulse Ctrl+S o use Archivo > Guardar informe....

El dialogo de guardado ofrece informes TXT y HTML. TXT agrupa por tipo de lectura y dispositivo, para que los nombres largos de hardware se escriban una vez como encabezados. HTML usa tablas con sensor, valor y fuente.

El primer guardado crea la carpeta Reports si no existe. Para compartir un informe, envie el archivo .html o .txt guardado. HTML suele ser mas comodo de leer en un navegador; TXT es util para pegar en mensajes. Si alguien necesita un paquete de soporte mas completo, use Ayuda > Ejecutar diagnosticos...; los diagnosticos incluyen ambos formatos, registros y un resumen en un ZIP.

Abrir el informe de otro equipo

Pulse Ctrl+O o use Archivo > Abrir informe... para cargar un informe TXT o HTML guardado por Sensor Readout. Si alguien envia un informe dentro de un archivo ZIP, abra el ZIP directamente y Sensor Readout usara el primer informe legible que encuentre dentro.

El informe se abre como una instantanea estatica en la vista normal de categorias y arbol. Asi puede inspeccionar el equipo de otra persona como si fuera su vista actual. El modo de informe estatico no actualiza valores en vivo, no ejecuta alarmas y no controla hardware. Solo muestra los datos guardados en el informe.

Los detalles y comandos de copia siguen funcionando cuando el informe contiene los campos necesarios. Esto ayuda cuando alguien envia datos USB, audio, pantalla, red, ventiladores, bateria o SMART y usted quiere revisar filas concretas sin leer todo el archivo manualmente.

Use Archivo > Volver a lecturas en vivo o Ctrl+R para volver al equipo actual. Los informes HTML guardados por Sensor Readout 2.0 o posterior incluyen una instantanea oculta legible por la app para una mejor importacion. Los informes TXT permanecen como texto normal y se abren lo mejor posible.

Transferencia de configuracion

Use Archivo > Exportar configuracion y perfiles... o Ctrl+E para crear un paquete de configuracion. Sensor Readout muestra primero una lista de comprobacion, para que pueda elegir exactamente que incluir. Use Archivo > Importar configuracion y perfiles... o Ctrl+Mayus+I para recuperar configuracion.
